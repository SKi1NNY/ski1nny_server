# Ingredient Knowledge Base

성분 설명 RAG에 사용할 원문 문서를 저장하는 폴더입니다.

작성 규칙:

- 성분 하나당 문서 하나
- 파일명은 영문 slug 사용
- 모든 섹션은 Markdown heading으로 작성
- 긴 문단보다 짧은 bullet 위주로 작성
- 아래 섹션 순서를 기본으로 유지
- 출처는 가능한 한 구체적으로 남기기

권장 섹션:

1. 성분명
2. 한글명
3. 효능
4. 작용 원리
5. 주의사항
6. 추천 피부 타입
7. 비추천/주의 피부 타입
8. 사용 팁
9. 근거 수준
10. 출처
